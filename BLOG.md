# Automate Ronin Blockchain Operations with Our New n8n Community Node

We're excited to announce the release of n8n-nodes-ronin-axie, a powerful new community node that brings Ronin blockchain automation to n8n workflows!

## The Challenge

As Web3 gaming and NFT ecosystems continue to expand, developers and businesses working with Axie Infinity, Ronin-based NFTs, Katana DEX, and Mavis Hub face a common challenge: automating blockchain interactions without writing complex custom code. Whether you're managing NFT collections, monitoring DEX transactions, or building gaming infrastructure, repetitive blockchain operations consume valuable development time.

## Meet n8n-nodes-ronin-axie

Velocity BPA has developed this specialized node to bridge the gap between n8n's powerful automation platform and the Ronin blockchain ecosystem. Now you can build sophisticated workflows that interact with Axie Infinity, trade on Katana DEX, manage NFTs, and integrate with Mavis Hub gaming—all through n8n's intuitive visual interface.

## Key Features

- **Axie Infinity Integration**: Automate game-related operations and data retrieval
- **NFT Management**: Streamline NFT minting, transfers, and collection monitoring
- **Katana DEX Operations**: Execute and monitor decentralized exchange activities
- **Mavis Hub Gaming**: Connect with the broader Ronin gaming ecosystem
- **Blockchain Interactions**: Direct access to Ronin blockchain functionality

## Quick Start

Getting started is simple. Install the node directly in your n8n instance:


npm install n8n-nodes-ronin-axie


Once installed, you'll find the Ronin-Axie node available in your n8n node palette, ready to drag and drop into your workflows.
